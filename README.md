# Shooter

Engineering-Grade QGIS Plugin for Sector Geometry & RF Validation

Shooter is a professional QGIS plugin designed for RF engineers and GIS analysts to manage sector geometry, azimuth validation, and structured site visualization.

## Features
- Add Site Tool
- Add Sector Tool
- Sector Geometry Builder
- Azimuth Control
- Engineering-based validation

## Requirements
- QGIS 3.x
- Python 3
